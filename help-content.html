Description

Welcome to the Ultimate Score Keeping app for board games or card games or any game that you would normally need to use a pencil and paper. Very simple to use, just enter the name of each player and enter the score as you go. The app will keep track of the number of rounds that were played, add up the score, sort and rank players by either low or high score.

Tip: You can edit player's color, player's name, add players or delete players anytime.

Getting Started

Once you have entered all the players names, you can now enter their score for each round of play.

Do this by touching the “+” field and entering their score. If you want to use the calculator function for the player's score, touch the “CALC” button.

Only after you have entered a score for each player, touch the + Submit Scores button to create a round of play.

Tip: Do NOT Submit Score until you have entered all the player's scores. Otherwise, you will create a Round each time you touch Submit.

Display Options

There are two types of display options
  • LIGHT and DARK mode which is toggled with 🌙.
  • LEADER BOARD and TABLE SEATING mode which is toggled with🪑.
🏆 LEADER BOARD display

Displays each player ranked by their score

🪑 TABLE SEATING display

Displays each player in seating order along with their ranking

Note: When in TABLE SEATING mode, players can be moved to the order in which they are seated around the table.

Uncheck the 'SEATING LOCKED' button in order to move players.

⚠️ IMPORTANT! When finished moving players, make sure the 'SEATING LOCKED' option button is checked.

This is important because if you don’t do this, you can accidently move players and/or clear scores while you are trying to enter them. This will cause unneeded frustration.

Saving Games

Update Current (or Change Name) : This will save the game with the current name, date, and time. Or you can change the name which will overwrite the current name.

Save Copy (or Enter New Name) : This will save a copy of the game with the current date and time or you can change the name and save the game.

Additional Features

  • Player’s name can be edited by touching their name
  • Color of player can be changed by touching their color
  • Can delete a player by touching the ‘trash can’
  • LAST ROUND option button to show or hide last round scores
  • Scores for each round are tracked in the ROUND HISTORY at the bottom of screen
  • Can delete a round, or touch a score in the round to edit it
